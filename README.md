# 🏛️ Minecraft Server Core Archive — Эпоха моддинга

**Добро пожаловать в музей серверных ядер Minecraft!**  
Этот репозиторий — архивная коллекция гибридных серверных сборок, которые охватывают почти всю историю моддинга — от легендарной **Beta 1.7.3** до современных версий **1.21.1**.

Здесь собраны **релизы**, которые когда-то произвели революцию: они позволили одновременно использовать моды (Forge) и плагины (Bukkit/Spigot) на одном сервере. Благодаря им появились тысячи уникальных проектов, многие из которых живы до сих пор.

> **⚠️ Внимание:** Все файлы предоставлены **как есть** (AS IS) исключительно в архивных и образовательных целях. Оригинальные разработчики (MCPC+, Cauldron, CatServer, Arclight, Lightfall, Babric) сохраняют все права на свои творения. Мы лишь сохраняем их наследие.

## 🛠️ Инструкция по использованию

### 1️⃣ Выберите версию Java
- Для **MCPC+/Cauldron (1.4.7 – 1.7.10)** и **CatServer 1.12.2** используйте **Java 8** (JDK 8u101 или выше).  
- Для **Arclight (1.16.5+)** требуется **Java 17** или новее.

### 2️⃣ Подготовка библиотек
Многие ядра требуют дополнительные библиотеки (зависимости Forge и Bukkit).  
Распакуйте соответствующий архив `Библиотеки *.zip` в папку **`libraries/`** рядом с JAR-файлом.
